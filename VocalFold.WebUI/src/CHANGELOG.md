# Changelog

All notable changes to this project will be documented in this file.

## [1.3.0] - 2025-11-20

### Added
- Executable Commands feature with application launcher functionality
- Usage tracking for commands and keyword replacements


## [1.2.3] - 2025-11-06

### Added
- Automatic Releases on GitHub when a new Version is tagged


## [1.1.0] - 2025-11-03

### Added
- Microphone Settings page with device selection and testing capabilities
- Real-time audio level monitoring and visualization
- Microphone test functionality with transcription testing
- Changelog page to track version history
- External configuration file support for keywords (for GDrive/OneDrive/etc. sync)

### Changed
- Improved settings UI with file selection for keywords path
- Enhanced AudioRecorder with level and spectrum callbacks
- Updated SystemSettings to integrate microphone configuration
- Reduced verbosity of Fast Fourier Transform log output

### Fixed
- HTTP 500 error when saving settings changes
- Settings API error handling and response serialization

---

## [1.0.0] - 2025-11-01
### Added
- Initial public release
- Voice-to-text transcription with Whisper
- Keyword replacement system with categories
- Hotkey support for quick voice input
- System tray integration
- Auto-start with Windows option
- General and system settings management
